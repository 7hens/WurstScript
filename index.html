WurstScript

A programming language which compiles to Jass (the language to create Warcraft III Maps).

Wurstscript is an imperative, object-oriented, statically-typed, beginner-friendly programming language with significant whitespace and a readable syntax.

Wurst Pack

The Wurst Pack contains a version of the editor with the Wurst compiler built in.

You can always download the latest build from our build server.

  • Complete Pack: This is the complete pack. You usually should download this, but you can use the other downloads if you only want to update parts of the Wurst Pack.
  • Compiler: This download only contains the wurstscript folder of the Pack. Usually it is sufficient to update this folder, since other parts of the Pack are rarely changed.
  • Standard Library: This is standard library.
  • Wurstscript.jar: This is only the main jar file for the compiler. For small bugfixes it is often sufficient to replace this file, but to be sure you should usually choose the bigger downloads.
