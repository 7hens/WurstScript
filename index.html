WurstScript

A programming language which compiles to Jass (the language to create Warcraft III Maps).

Wurstscript is an imperative, object-oriented, statically-typed, beginner-friendly programming language with significant whitespace and a readable syntax.

Installation Notes (EN)

Eclipse Plugin

You can install the Eclipse plugin using the update site at http://sunayama.de/WurstUpdt/site/. In Eclipse go to ""Help - Install New Software" and add the update site mentioned above.

Wurst Pack

The Wurst Pack contains a version of the editor with the Wurst compiler built in. You can download the Updater from this 2 links (See installation notes for more info): Wurst Pack (32bit) Wurst Pack (64bit) If you are a more advanced user and always want a bleeding edge version you can build it from the sources using the build-jar.xml file which you can find in the compiler project "de.peeeq.wurstscript".

Source Code

You can download the sources for this project in either zip or tar formats.
